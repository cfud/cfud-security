Сканер AV сканирует указанную директорию и выявляет изменения с момента последнего сканирования.

Для использования сканера необходимо поместить av.php в любую директорию сервера. Но перед этим
необходимо произвести первоначальную настройку сканера, для этого откройте файл av.php и укажите
необходимые для вас настройки. 

В переменной $homeDir необходимо указать директорию, которая и будет сканироваться на все изменения.
Для корректной работы требуется указывать ПОЛНЫЙ путь до директории.

$ignoreDirs содержит массив директорий, которые будут игнорироваться сканером, можно так же указывать
пути до файлов. Для корректной работы требуется указывать ПОЛНЫЙ путь до директорий и файлов.

В $reportsDir можно указать ПОЛНЫЙ путь до директории, в которую будут записываться текстовые файлы с
отчетами о проверка. Если поставить значение null, то будет создана директория av рядом с файлом av.php

Отчеты сохраняются в тектовые файлы с именами в формате report_ГОД-МЕСЯЦ-ЧИСЛО_ЧАС-МИНУТА-СЕКУНДА.txt
Если имя файла содержит пометку CLEAN, то это означает, что в этот момент времени изменений обнаружено небыло.

Для того чтобы запустить сканер, можно установить задание CRON с нужной переодичностью на исполнение файла av.php
или обращаться прямо на av.php для ручного сканирования. 

av.php не подает на вывод никаких сообщений и результатов работы, за исключением отчетов в текстовых файлах.
